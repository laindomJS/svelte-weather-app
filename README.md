
# Weather App

This is a weather app developed in Svelte with TailwindCSS and the API of weatherapi.
## Tech Stack

**Client:** Svelte, TailwindCSS

**API:** [weatherapi](https://www.weatherapi.com/)

**Build:** Vite

**Libraries:** dayjs
